Exact Differential Equations

Virtual Lab for Engineering Mathematics

Aim of the Experiment

Primary Objective

To understand the concept of exact differential equations and learn systematic methods to solve them step-by-step.

Learning Outcomes

  • Identify exact differential equations
  • Apply the condition for exactness: ∂M/∂y = ∂N/∂x
  • Solve exact differential equations systematically
  • Handle non-exact equations using integrating factors

Skills Developed

  • Mathematical reasoning and problem-solving
  • Step-by-step analytical thinking
  • Verification of solutions
  • Application of partial derivatives

Sample Exact Differential Equation

(2xy + y²)dx + (x² + 2xy)dy = 0

M(x,y) = 2xy + y², N(x,y) = x² + 2xy

∂M/∂y = 2x + 2y = ∂N/∂x ✓

Theoretical Background

Definition

A differential equation of the form M(x,y)dx + N(x,y)dy = 0 is called exact if there exists a function ψ(x,y) such that:

dψ = M(x,y)dx + N(x,y)dy

This implies: ∂ψ/∂x = M(x,y) and ∂ψ/∂y = N(x,y)

Condition for Exactness

The necessary and sufficient condition for exactness is:

∂M/∂y = ∂N/∂x

Solution Procedure

1

Verify exactness: Check if ∂M/∂y = ∂N/∂x

2

Integrate M w.r.t x: ψ = ∫M dx + g(y)

3

Differentiate ψ w.r.t y and compare with N

4

Find g'(y) and integrate to get g(y)

5

Write general solution: ψ(x,y) = C

Integrating Factors

If ∂M/∂y ≠ ∂N/∂x, the equation is not exact. An integrating factor μ can sometimes make it exact:

  • If (∂M/∂y − ∂N/∂x)/N = f(x) only → μ = e^(∫f(x)dx)
  • If (∂N/∂x − ∂M/∂y)/M = g(y) only → μ = e^(∫g(y)dy)
